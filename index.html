Checklist

Design

  • Body Copy is NOT centered
  • Max ~70 characters per line
  • Consistent padding for similar elements
  • Images = NOT pixelated

Sass

  • “burbon install” in stylesheet folder EVERY TIME
  • Change style.css to stylesheet.scss
  • Run “sass –watch stylesheet/styles.scss”
  • Inception Rule: after 4 levels the dream collapses
  • Run Sass Beautify to take care of spaces

HTML and CSS

  • Container div
  • Meta-charset: utf-8
  • Minimal "br" tags
  • No floating divs
  • All div tags are closed
  • ResetCSS is BEFORE StyleCSS
  • StyleCSS functions across all pages
  • Alts for the images
  • Clear(right) after floating
  • THE CODE IS MINIMAL
  • Run HTML Beautify to take care of spaces

Git

  • ADD ⇒ COMMIT ⇒ PUSH
  • checkout gh-pages ⇒ pull origin master ⇒ push
  • “python –m SimpleHTTPServer” makes a local server to see changes in real time

Responsiveness

  • Always do mobile first
  • Include "< meta name="viewport" content="width=device-width, initial-scale=1.0" >"
  • Save images well
